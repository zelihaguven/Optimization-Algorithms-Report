# Metaheuristic Optimization Algorithms: PSO, DE, SA, and TS

This repository is created for our academic project analyzing and comparing four metaheuristic algorithms: Particle Swarm Optimization (PSO), Differential Evolution (DE), Simulated Annealing (SA), and Tabu Search (TS).

## Contents

- **Simple Examples:** Small-scale applications demonstrating the core logic of each algorithm.
- **Realistic Simulation Examples:** Simulations for testing algorithm performance in more complex and realistic scenarios.
- **Analysis and Comparison Codes:** Python scripts for further analyses and performance comparisons.
- **Academic Paper (English):** The English academic version of the study, including methodology, analyses, and results.

# Meta-Sezgisel Optimizasyon Algoritmaları: PSO, DE, SA ve TS

Bu depo, dört meta-sezgisel algoritmanın (Parçacık Sürü Optimizasyonu - PSO, Diferansiyel Evrim - DE, Benzetilmiş Tavlama - SA ve Tabu Arama - TS) analiz ve karşılaştırmasını içeren akademik çalışmamız için hazırlanmıştır.

## İçerik

- **Basit Örnekler:** Algoritmaların temel mantığını göstermek için küçük ölçekli uygulamalar.
- **Gerçek Simülasyon Örnekleri:** Algoritmaların daha karmaşık ve gerçekçi senaryolardaki performansını test etmek için hazırlanmış simülasyonlar.
- **Analiz ve Karşılaştırma Kodları:** İlerleyen analizlerde ve performans karşılaştırmalarında kullanılacak Python kodları.
- **Akademik Makale (İngilizce):** Çalışmanın İngilizce akademik versiyonu; metodoloji, analizler ve sonuçları içerir.
